# To do App

To do app pake next js + mongoDB

## Getting Started

### Prerequisite

- Node
- NPM

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/nicholass25/TubesARC1.git
   ```

2. Change directory

   ```sh
   cd TubesARC1
   ```

3. Install Dependencies
   ```sh
   npm install
   ```

## Usage

1. Run the app

   ```sh
   npm run dev
   ```
